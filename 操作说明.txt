# Need run in fish (/opt/bin/fish), needs git.
# 在家里NAS上的终端，用nano编辑无法输入中文，但通过micro可以

run bash docker-login.sh to login first, password see KeePassX.

Add your docker image name which want to pull to images.txt,
then run :
  git add images.txt
  git commit -m "some comment"
  git push

 (P.S remote repo is git@github.com:wildtang/docker_images_sync.git)

Then run command like below to pull the image:
  ./cmd/docker_pull redimp/otterwiki:2
